# Healthcare Data Analysis – Diabetes Dataset

## 📌 Objective
Analyze healthcare data to identify patterns and risk factors associated with diabetes.

## 🛠️ Tools Used
- Python (Pandas, Matplotlib, Seaborn)

## 📊 Project Workflow
- Data loading and cleaning
- Exploratory Data Analysis (EDA)
- Data visualization using Python

## 📈 Key Insights
- Higher glucose levels are strongly linked to diabetes
- BMI and age contribute to increased risk
- Majority of patients are non-diabetic, but high-risk groups exist

## 📷 Dashboard
The project includes a Python-based dashboard showing:
- Diabetes distribution
- BMI trends
- Age vs Glucose relationship

## 🚀 Conclusion
This project demonstrates an end-to-end data analysis workflow in the healthcare domain using Python.
Healthcare data analysis using Python (Pandas, Seaborn) to identify diabetes risk factors through EDA and dashboard visualization.
